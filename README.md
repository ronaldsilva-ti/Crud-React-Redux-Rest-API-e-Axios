# CRUD - React, Redux, Rest API e Axios

#Local Instalação

1.Install Node.js

2.```git clone https://github.com/ronaldsilva-ti/Crud-React-Redux-Rest-API-e-Axios.git```

3.```cd Crud-React-Redux-Rest-API-e-Axios```

4.```npm install```

5.```npm start ```
